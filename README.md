# Data-Course-ES

Curso introductorio de programación y análisis de datos con Python, diseñado para aprender de forma **práctica, progresiva e interactiva**.  
El hilo conductor del curso es un **proyecto realista**: construir paso a paso un **gestor de finanzas personales**.

## 📦 Estructura del curso

```plaintext
📁 00_bienvenida_y_objetivos         → Introducción general
├── 01_bienvenida.md                    → Presentación del curso
├── 02_intro_proyecto_finanzas.md      → Explicación del proyecto principal

📁 01_modulo_fundamentos            → Fundamentos de programación en Python, crear un pequeño script que almacene transacciones simples y calcule el balance. input/output, tipos, estructuras, funciones.
├── 01_python_basico.md                → Qué es Python, cómo se usa en proyectos reales
├── 02_datos_y_estructuras.md          → Tipos de datos y estructuras aplicadas a finanzas
├── 03_bucles.md                       → Uso de bucles para revisar transacciones
├── 04_funciones.md                    → Funciones para automatizar cálculos
